# Igeragozás (Konjugation)

## 1. Gyenge igék (Szabályos ragozás)

A legtöbb német ige így ragozódik. Az `-en` végződést levágjuk, és helyére illesztjük a ragokat.

**Példa:** *lernen* (tanulni) $\to$ *lern-*

| **Személy**         | **Rag** | **Példa**      |
| ------------------- | ------- | -------------- |
| **ich** (én)        | **-e**  | ich lern**e**  |
| **du** (te)         | **-st** | du lern**st**  |
| **er/sie/es** (ő)   | **-t**  | er lern**t**   |
| **wir** (mi)        | **-en** | wir lern**en** |
| **ihr** (ti)        | **-t**  | ihr lern**t**  |
| **sie/Sie** (ők/Ön) | **-en** | sie lern**en** |

## 2. Rendhagyó igék (Tőhangváltós igék)

Vannak igék, amiknél az **E/2.** és **E/3.** személyben megváltozik a tőhangzó. A ragok ugyanazok maradnak!

### A) e $\to$ i / ie

**Példa:** *helfen*

| **Személy**         | **Példa**      |
| ------------------- | -------------- |
| **ich** (én)        | ich helfe      |
| **du** (te)         | du hilfst      |
| **er/sie/es** (ő)   | er hilft       |
| **wir** (mi)        | wir helfen     |
| **ihr** (ti)        | ihr lern**t**  |
| **sie/Sie** (ők/Ön) | sie lern**en** |
