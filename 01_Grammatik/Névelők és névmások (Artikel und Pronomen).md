# Névelők és névmások (Artikel und Pronomen)

## Nyelvtani esetek (Kasus)

|**Eset neve (latinul)**|**Magyar megnevezés**|**Kérdőszó**|
|---|---|---|
|**Nominativ**|Alanyeset|Wer? Was? (Ki? Mi?)|
|**Akkusativ**|Tárgyeset|Wen? Was? (Kit? Mit?)|
|**Dativ**|Részes eset|Wem? (Kinek? Minek?)|
|**Genitiv**|Birtokos eset|Wessen? (Kié? Minek a...?)|

## I. Névelők

A németben a főnév neme (hímnem, nőnem, semleges) és esete határozza meg a névelő alakját.

### 1. Határozott és határozatlan névelők

A határozatlan névelőnek (_ein/eine_) nincs többes száma, ilyenkor a főnév névelő nélkül áll.

| **Eset**      | **Hímnem (m)**        | **Nőnem (f)**   | **Semleges (n)**      | **Többes szám (Pl.)** |
| ------------- | --------------------- | --------------- | --------------------- | --------------------- |
| **Nominativ** | der / ein             | die / eine      | das / ein             | die / -               |
| **Akkusativ** | **den / einen**       | die / eine      | das / ein             | die / -               |
| **Dativ**     | **dem / einem**       | **der / einer** | **dem / einem**       | **den + -n (rag)**    |
| **Genitiv**   | **des (+ -s/es) / -** | **der / -**     | **des (+ -s/es) / -** | **der / -**           |

### 2. Tagadó névelő (Kein / Keine)

Pontosan úgy ragozzuk, mint az _ein / eine_ párost, de van többes száma is.
- _Ich habe **keinen** Hund._ (Nincs kutyám - hímnem, Akkusativ)
- _Ich habe **keine** Kinder._ (Nincsenek gyerekeim - többes szám)

Ezek helyettesítik a főneveket, hogy ne kelljen ismételni őket.

## III. Birtokos névmások (Possessivpronomen)

A birtokos névmásnak két dolgot kell kifejeznie: **ki a birtokos** (szótő) és **mi a birtok neme/esete** (végződés).

### 1. A szótövek (Kié?)

- én → **mein-**
- te → **dein-**
- ő (hn./sn.) → **sein-**
- ő (nn.) / ők / Ön → **ihr-**
- mi → **unser-**
- ti → **euer-**
  Ha az **euer** szó végződést kap, a szótő utolsó **-e-** betűje kiesik!
	- _euer_ + _e_ → **eure**
	- _euer_ + _en_ → **euren**

### 2. A végződések

A birtokos névmásokat **úgy ragozzuk, mint az "ein" határozatlan névelőt**.
- _Das ist **mein** Vater._ (Alany, hímnem)
- _Ich liebe **meinen** Vater._ (Tárgy, hímnem → **-en** végződés)
- _Ich helfe **meiner** Mutter._ (Részes, nőnem → **-er** végződés)

## IV. Mutató névmások (Demonstrativpronomen)

A leggyakoribb a **dieser / diese / dieses** (ez a...). Úgy ragozzuk, mint a **határozott névelőt** (_der/die/das_).
- **Hímnem:** _dieser_ (Nom.), _diesen_ (Akk.), _diesem_ (Dat.)
- **Nőnem:** _diese_ (Nom.), _diese_ (Akk.), _dieser_ (Dat.)
- **Semleges:** _dieses_ (Nom.), _dieses_ (Akk.), _diesem_ (Dat.)
