## C) Zeitangaben (Időhatározók)

Ezek a szavak a „jelzőtáblák”, amik megmutatják, melyik igeidőbe tedd a mondatot.

| Ausdruck           | Zeitform             | Bedeutung          |
| ------------------ | -------------------- | ------------------ |
| **jetzt**          | Präsens              | most               |
| **heute**          | Präsens / Perfekt    | ma                 |
| **gestern**        | Perfekt / Präteritum | tegnap             |
| **früher**         | Präteritum           | régebben           |
| **letztes Jahr**   | Perfekt / Präteritum | tavaly             |
| **morgen**         | Präsens / Futur I    | holnap             |
| **bald**           | Futur I              | hamarosan          |
| **bis morgen**     | Futur II             | holnapra (már)     |
| **in zwei Jahren** | Futur II             | két év múlva (már) |
