## B) Positions- und Richtungsverben (Helyzetet és irányt kifejező igék)

| Frage      | Kasus         | Verb        | Erklärung           |
| ---------- | ------------- | ----------- | ------------------- |
| **Wohin?** | **Akkusativ** | **stellen** | állít (vmit)        |
|            |               | **legen**   | fektet, tesz (vmit) |
|            |               | **setzen**  | ültet (vkit/vmit)   |
|            |               | **hängen**  | felakaszt (vmit)    |

|            |               | **liegen**  | fekszik (vmi)       |
|            |               | **sitzen**  | ül (vmi)            |
|            |               | **hängen**  | lóg, függ (vmi)     |
