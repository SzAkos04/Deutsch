# Artikel und Pronomen

## A) Grammatische Begriffe (Nyelvtani fogalmak)

| Fachbegriff      | Bedeutung        |
| ---------------- | ---------------- |
| **der Kasus**    | eset (nyelvtani) |
| **der Artikel**  | névelő           |
| **das Pronomen** | névmás           |
| **Singular**     | egyes szám       |
| **Plural**       | többes szám      |
| **maskulin**     | hímnem           |
| **feminin**      | nőnem            |
| **neutral**      | semlegesnem      |
| **Nominativ**    | alanyeset        |
| **Akkusativ**    | tárgyeset        |
| **Dativ**        | részes eset      |
| **Genitiv**      | birtokos eset    |

## E) Personal- und Possessivpronomen (Névmások)

| Pronomen              | Bedeutung                           |
| --------------------- | ----------------------------------- |
| **ich / mich / mir**  | én / engem / nekem                  |
| **du / dich / dir**   | te / téged / neked                  |
| **er / ihn / ihm**    | ő (h) / őt / neki                   |
| **sie / sie / ihr**   | ő (n) / őt / neki                   |
| **es / es / ihm**     | ő (s) / őt / neki                   |
| **wir / uns / uns**   | mi / minket / nekünk                |
| **ihr / euch / euch** | ti / titeket / nektek               |
| **sie / sie / ihnen** | ők / őket / nekik                   |
| **Sie / Sie / Ihnen** | Ön / Önt / Önnek                    |
| **mein-**             | az én ...-m (enyém)                 |
| **dein-**             | a te ...-d (tiéd)                   |
| **sein-**             | az ő ...-ja (övé - hímnem/semleges) |
| **ihr-**              | az ő ...-ja (övé - nőnem)           |
| **unser-**            | a mi ...-unk (mienk)                |
| **euer-**             | a ti ...-otok (tiétek)              |


## F) Deklination der Artikel (Névelők ragozása)

| Kasus         | Maskulin (der) | Feminin (die) | Neutral (das) | Plural (die) |
| ------------- | -------------- | ------------- | ------------- | ------------ |
| **Nominativ** | der / ein      | die / eine    | das / ein     | die / -      |
| **Akkusativ** | den / einen    | die / eine    | das / ein     | die / -      |
| **Dativ**     | dem / einem    | der / einer   | dem / einem   | den ...-n    |
| **Genitiv**   | des ...-s      | der           | des ...-s     | der          |
